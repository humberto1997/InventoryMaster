InventoryMaster es una solución **Full-Stack** completa para el control de inventarios. El proyecto integra una API robusta construida en .NET con una interfaz de usuario dinámica que permite la gestión de stock en tiempo real.

## 🚀 Funcionalidades Clave

- **Gestión CRUD Completa**: Control total sobre el catálogo de productos (Crear, Leer, Actualizar, Eliminar).
- **Venta de Productos**: Sistema de transacciones que descuenta stock automáticamente con validaciones de disponibilidad.
- **Dashboard de Métricas**: Visualización instantánea del valor total del inventario, unidades en existencia y alertas de stock crítico.
- **Búsqueda Inteligente**: Filtrado dinámico en el frontend para localizar productos rápidamente.
- **Validaciones de Seguridad**: Protección de integridad de datos implementada tanto en el cliente (JS) como en el servidor (C#).

## 🛠️ Stack Tecnológico

- **Backend**: ASP.NET Core Web API (.NET 8) y Entity Framework Core.
- **Base de Datos**: Microsoft SQL Server.
- **Frontend**: HTML5, CSS3 y JavaScript Moderno (Vanilla).
- **Documentación**: Swagger UI para pruebas de endpoints.
